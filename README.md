# Netflix Movies and TV Shows Data Analysis

## Objective
This project analyzes Netflix movies and TV shows using a Kaggle dataset.

## Contents
- Data loading and cleaning
- Exploratory data analysis
- Genre, popularity, rating, language and release-year analysis
- Content-based recommendation system
- Optional popularity prediction baseline

## How to run
pip install -r requirements.txt
jupyter notebook notebooks/netflix_challenge_notebook.ipynb
